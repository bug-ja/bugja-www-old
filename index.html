ユーザサポートに関しては、将来の bug-ja.org の発展をお待ちください。 現在、ユーザ向けサポートメーリングリストの設置を計画中です。
なお、IRCもご利用になれます。 Bugzilla に関する日本語でのバグ報告・議論については、 Bugzilla@jpmozをご利用ください。
Bugzilla 日本語版開発に興味のある方は メーリングリスト にもぜひご登録ください。 また、bug-ja.org の svn レポジトリへのアクセス権限についても、 Bugzilla@jpmoz までご連絡ください。 (bug-jaプロダクトにバグとして登録してください。 ご協力いただける方にはどなたでもお渡しします。)

バージョン情報

  • bugzilla.org (本家) 分 : trunk = 4.5です
    • 4.5 : 現在の trunk です。
    • 4.4 : 現在の安定版です。
    • 4.2 : 一つ前安定版です。
    • 4.0 : ふるいの安定版です。セキュリティー修正のみが行われます。
    • - 3.6 : 古い安定版です。一切のサポートが行われません。
  • bug-ja 分 (日本語テンプレートセット)
    • 4.5 (trunk) : 不定期に更新され、アーカイブでなくsvnレポジトリとして公開されます
    • 4.0, 4.2, 4.4 : 通常通りサポートされます
    • 3.0, 3.2, 3.4, 3.6 : 過去のリリースです。メンテナンスされません。最新版へのアップグレードを行ってください。

bug-jaの一つの目的である、国際化に関するパッチに関しては、最新の安定板以降のみ を対象としてメンテナンスを行う予定にしていますます。
3.6, 3.4を利用されている方は、最新版へのアップグレードを強く推奨します。
